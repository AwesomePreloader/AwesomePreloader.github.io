The implementation:

Include AwesomeFrontend CSS and JS before the closing </body> (you don't need jQuery):


    	<link type="text/css" rel="stylesheet" href="css/awesomefrontend.css" />
    	<script type="text/javascript" src="js/awesomefrontend.js"></script>

The mobile menu declaration:


	<div class="mobile-menu" animation="slideIn" animation-duration="0.5s"></div>

The animation attribute sets the type of animation you want. Just type 'fadeIn', or 'slideIn' and you're ready to go.

The animation-duration sets the duration of the selected animation. Just type the duration in a Ns format (0.5, for example).

All anmiations are packed up with default speeds in case you don't want to set this value.
